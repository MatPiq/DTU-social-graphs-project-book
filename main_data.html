Data DescriptionÂ¶

For our project, we collected wikipedia articles corresponding to five major social science diciplines: Economics, Political Science, Anthropology, Sociology and Psychology. To do this we used the tool petscan which enables one to find all the sub-categories/articles corresponding to a given page. Petscan can be access both through a web-interface or programmatically through for example Python. We did it programmatically by querying the main page corresponding to each discipline.
